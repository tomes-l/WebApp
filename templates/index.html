About the App:

  • The program only take in input the following organic atoms: H, B, C, N, O, P, S, F, Cl, Br, I

  • The SMILES given can include all the symbols used in the notation (i.e. =, #, (, ), [, ], :, "small caps for cycle", \, /, @, @@, +, -)

  • The file given in input must be a mol file (i.e. aspirine.mol)

  • 'Atom identifier' is a number given to a certain atom in a given molecule (refer to the mol file)
